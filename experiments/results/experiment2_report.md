## 분석 및 인사이트

### MCP Tool의 효과
- **① MCP 없이**: LLM이 자체 지식으로만 답변 → 실시간 데이터(날씨, 시간) 부정확, OCR 불가
- **② 정상 MCP**: Tool을 통해 정확한 실시간 데이터 제공 → 정확도와 신뢰성 향상
- **③ 망가뜨린 MCP**: Tool 실패 시 LLM이 오류를 처리하지만 결과 품질 저하
